### 1. Numerical solution

Write a numerical code in C++ which :

- Solve the system of equations for different input parameters $(\alpha,beta,gamma,delta,a,b)$ using a Runge-Kutta scheme. 
- For some set of parameters check the results obtained in the original paper by Chua [Chua 1993] by producing some plots of the the famous **double-scroll attractor**.
